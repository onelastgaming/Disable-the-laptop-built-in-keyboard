# Disable-the-laptop-built-in-keyboard
快速禁用笔记本电脑键盘
笔记本内置键盘一键管理工具
Laptop Built-in Keyboard One-Click Manager
项目介绍
本工具是基于 Windows 批处理脚本编译的 EXE 程序，通过修改 i8042prt 系统服务，一键控制笔记本内置键盘的启用 / 禁用状态，无需手动操作命令行或设备管理器，小白也能轻松使用。
核心功能
自动检测当前内置键盘服务状态
一键切换：自动 → 手动 / 禁用 → 手动
双重确认机制，防止误操作
管理员权限自动校验，提升稳定性
适用系统
✅ Windows 10 / 11（全版本）✅ 所有搭载 PS/2 标准键盘的笔记本
使用方法
1. 运行前准备
⚠️ 重要：如需禁用内置键盘，必须先连接外接键盘，否则无法操作电脑
2. 启动程序
右键点击 exe 程序
选择 以管理员身份运行
程序会自动检测当前键盘服务状态
3. 操作说明
程序会根据当前状态自动提供对应选项，输入 Y 确认，N 取消，按回车执行
自动启动 → 可改为手动启动（禁用内置键盘）
禁用状态 → 可改为手动启动（恢复内置键盘）
手动启动 → 无需修改，正常使用
状态说明
表格
服务状态	含义	键盘效果
AUTO_START	自动启动	内置键盘正常使用
DEMAND_START	手动启动	内置键盘禁用失效
DISABLED	禁用	内置键盘完全禁用
常见问题
1. 运行提示 “请以管理员身份运行”
解决：右键程序 → 选择「以管理员身份运行」
2. 修改后键盘没变化
解决：重启电脑即可生效
3. 笔记本键盘无法恢复
解决：重新运行程序，将服务改回「手动启动」，重启电脑
4. 不支持我的笔记本
原因：部分新款笔记本使用非 PS/2 键盘，此工具不兼容
安全说明
本工具仅修改系统键盘服务配置，无任何恶意代码
不会修改系统文件、不会窃取数据
可随时恢复默认设置，对电脑无损害
脚本信息
原始脚本：批处理（.bat）
编译工具：Bat To Exe Converter
核心服务：i8042prt（Windows 键盘驱动服务）
